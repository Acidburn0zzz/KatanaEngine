#Project Katana#
The Katana project is a project that aims to develop a heavily modified version of the FitzQuake engine that has been specifically developed for our game, **OpenKatana**.

###New Features###
- SDL2
- Platform library
- Point-based planar shadows
- Blob shadows
- MD2 support
- PNG support
- Rotating brushes
- Bug fixes...
